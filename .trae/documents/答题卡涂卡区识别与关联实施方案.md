# 答题卡涂卡区识别与关联实施方案

## 🎯 目标

从试卷切片数据中识别涂卡区，单独存储并用 OpenCV 识别填涂答案，最终与题目关联。

---

## 📋 需求分析

### 1. 输入数据
- 内容提取步骤输出：`04_content_output.json`
- 包含大模型识别的所有区域（题目、答案、涂卡区）
- 涂卡区特征：
  - `number` 字段为范围格式（如 `"1-21"`）
  - `description` 包含"涂卡区"、"填涂"、"选择题"等关键字
  - 位于答题纸上（`sheet_type: "answer_sheet"`）

### 2. 输出结果
- 涂卡区物理切片图片（单独文件夹）
- 每个题的填涂答案（JSON 格式）
- 填涂答案与题目切片的关联关系

### 步骤 1：识别涂卡区切片

**文件**：`preprocessor/src/tasks/task_extract_answer_card.py`（新建）

**功能**：
1. 读取 `04_content_output.json`
2. 遍历所有切片数据
3. 识别涂卡区的切片（通过 `number` 格式和 `description` 关键字）
4. 标记涂卡区切片

**识别逻辑**：
```python
def is_answer_card_area(fragment: dict) -> bool:
    """判断是否是涂卡区"""
    number = fragment.get('number', '')
    description = fragment.get('description', '')
    
    # 条件 1：number 包含连字符（如 "1-21"）
    has_range_format = '-' in str(number)
    
    # 条件 2：description 包含涂卡区关键字
    keywords = ['涂卡', '填涂', '选择题', '客观题', 'A/B/C/D']
    has_keyword = any(kw in description for kw in keywords)
    
    # 条件 3：位于答题纸上
    sheet_type = fragment.get('sheet_type', '')
    is_answer_sheet = sheet_type in ['answer_sheet', '答题纸', 'mixed', '混合纸']
    
    return (has_range_format or has_keyword) and is_answer_sheet
```

**输出**：
- 涂卡区切片列表
- 包含：坐标、图片路径、题号范围等

2. 根据坐标从原图中切片涂卡区域
3. 保存为单独的图片文件

**切片逻辑**：
```python
def crop_answer_card_area(fragment: dict, workspace_dir: str) -> str:
    """切片涂卡区并保存"""
    import cv2
    
    # 1. 获取原图路径和坐标
    image_path = fragment['source_corrected_image']
    points = fragment['points']
    
    # 2. 加载图片
    image = cv2.imread(image_path)
    h, w = image.shape[:2]
    
    # 3. 将归一化坐标转换为像素坐标
    x_min = int(min(points['top_left'][0], points['bottom_left'][0]) / 1000 * w)
    y_min = int(min(points['top_left'][1], points['top_right'][1]) / 1000 * h)
    x_max = int(max(points['top_right'][0], points['bottom_right'][0]) / 1000 * w)
    y_max = int(max(points['bottom_left'][1], points['bottom_right'][1]) / 1000 * h)
    
    # 4. 切片
    crop_image = image[y_min:y_max, x_min:x_max]
    
    # 5. 保存
    number = fragment.get('number', 'unknown').replace('/', '_')
    output_path = os.path.join(workspace_dir, 'answer_card_areas', f'answer_card_{number}.jpg')
    os.makedirs(os.path.dirname(output_path), exist_ok=True)
    cv2.imwrite(output_path, crop_image)
    
    return output_path
```

### 步骤 3：OpenCV 识别填涂答案

**文件**：`preprocessor/src/answer_card/ocr_detector.py`（新建模块）

**功能**：
1. 定位每个选项框的位置
2. 识别每个选项是否被填涂
3. 返回填涂结果

**实现方案**：

#### 3.1 选项框定位
```python
def detect_option_boxes(crop_image: np.ndarray) -> List[dict]:
    """
    检测每个选项框的位置
    
    返回：
    [
        {'question_num': '1', 'option': 'A', 'bbox': [x, y, w, h], 'filled': True/False},
        {'question_num': '1', 'option': 'B', 'bbox': [x, y, w, h], 'filled': True/False},
        ...
    ]
    """
    # 1. 灰度化 + 二值化
    gray = cv2.cvtColor(crop_image, cv2.COLOR_BGR2GRAY)
    _, thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)
    
    # 2. 查找轮廓
    contours, _ = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    # 3. 筛选选项框轮廓（根据面积、形状等）
    option_boxes = []
    for cnt in contours:
        area = cv2.contourArea(cnt)
        if 100 < area < 5000:  # 根据实际大小调整
            x, y, w, h = cv2.boundingRect(cnt)
            aspect_ratio = w / float(h)
            if 0.5 < aspect_ratio < 2.0:  # 接近圆形或方形
                option_boxes.append({'bbox': [x, y, w, h], 'contour': cnt})
    
    # 4. 按位置排序，分配题号和选项
    # TODO: 根据实际布局排序（横向/纵向）
    
    return option_boxes
```

#### 3.2 填涂检测
```python
def detect_filled_option(crop_image: np.ndarray, option_box: dict) -> bool:
    """
    判断选项是否被填涂
    
    方法：
    1. 分析框内像素密度
    2. 填涂的区域通常颜色更深、密度更高
    """
    x, y, w, h = option_box['bbox']
    roi = crop_image[y:y+h, x:x+w]
    
    # 1. 灰度化
    gray = cv2.cvtColor(roi, cv2.COLOR_BGR2GRAY)
    
    # 2. 计算暗色像素比例（填涂区域通常较暗）
    _, thresh = cv2.threshold(gray, 128, 255, cv2.THRESH_BINARY_INV)
    filled_ratio = cv2.countNonZero(thresh) / (w * h)
    
    # 3. 判断阈值（需要根据实际情况调整）
    return filled_ratio > 0.3  # 30% 以上被覆盖认为是填涂
```

#### 3.3 主识别函数
```python
def recognize_answer_card(crop_image_path: str) -> dict:
    """
    识别涂卡区的所有答案
    
    返回：
    {
        '1': 'A',  # 第 1 题选 A
        '2': 'C',  # 第 2 题选 C
        '3': 'B',
        ...
    }
    """
    import cv2
    
    # 1. 加载图片
    image = cv2.imread(crop_image_path)
    
    # 2. 检测所有选项框
    option_boxes = detect_option_boxes(image)
    
    # 3. 识别每个选项是否填涂
    answers = {}
    for box in option_boxes:
        question_num = box['question_num']
        option = box['option']  # 'A', 'B', 'C', 'D'
        
        is_filled = detect_filled_option(image, box)
        
        if is_filled:
            # 如果该题已经有答案，可能是多选或误识别
            if question_num in answers:
                # TODO: 处理多选情况
                pass
            else:
                answers[question_num] = option
    
    return answers
```

### 步骤 4：关联填涂答案与题目

**文件**：`preprocessor/src/tasks/task_link_answers.py`（新建）

**功能**：
1. 读取涂卡区识别结果
2. 读取题目切片数据
3. 按题号关联

**关联逻辑**：
```python
def link_answers_to_questions(
    answer_card_results: dict,  # 涂卡区识别结果
    merged_results: dict        # 合并后的题目数据
) -> dict:
    """
    将填涂答案与题目关联
    
    返回：
    {
        '1': {
            'question': {...},      # 题目切片信息
            'answer': 'A',          # 学生填涂的答案
            'answer_card_bbox': [...]  # 涂卡区坐标
        },
        '2': {
            'question': {...},
            'answer': 'C',
            'answer_card_bbox': [...]
        },
        ...
    }
    """
    linked_results = {}
    
    # 1. 遍历所有题号
    for question_number, answer in answer_card_results.items():
        # 2. 找到对应的题目
        if question_number in merged_results:
            question_fragments = merged_results[question_number]
            
            # 3. 找到题目纸上的题目（不是答题纸）
            question_fragment = None
            for fragment in question_fragments:
                if fragment.get('is_question_sheet', False):
                    question_fragment = fragment
                    break
            
            # 4. 关联
            if question_fragment:
                linked_results[question_number] = {
                    'question': question_fragment,
                    'answer': answer,
                    'answer_source': 'answer_card'
                }
    
    return linked_results
```
