# Changelog

## v0.3 (2026-03-20)

### ✨ Features
- **拼图测试工具稳定版**：完成长图分类功能的测试和优化
- **大模型长图分类**：实现基于拼接长图的页面类型分类，利用全局上下文提高准确率
- **套-张编号系统**：在透视矫正环节实现一套试卷的概念，生成唯一的套编号和张编号
- **配置化分类**：支持通过配置文件选择单页分类或长图分类
- **张顺序智能判断**：大模型根据试卷内容逻辑判断正确顺序，不依赖输入顺序

### 📁 目录结构
- 新增 `stitched_images/` 文件夹，专门存放拼装后的长图
- 新增配置文件系统，支持灵活配置分类方式

### 🚀 后续计划
- **拼图分类**：完善长图分类功能，支持更多试卷类型
- **切片功能**：基于套-张编号系统，实现题目和答案的切片与关联
- **完整单元形成**：将题目切片和答案切片组合成完整的题目-答案单元

### 🔧 技术改进
- 优化大模型提示词，提高分类准确率
- 实现数据结构支持后续切片功能
- 保留原始输入顺序作为参考，便于调试
